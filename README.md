Это тестовая версия программы для термогазодинамического расчета энергетической газотурбинной установки (ЭГТУ);
Описание термодинамических свойств воздуха и продуктов сгорания следующее:
- Есть таблица термодинамических свойств влажного воздуха;
- Есть таблица термодинамических свойств чистых продуктов сгорания метана;
Термодинамические свойства продуктов сгорания с учетом коэф-та избытка воздуха расчитываются.
Также присутствует модуль газодинамических функций.
Программа будет обновляться.
